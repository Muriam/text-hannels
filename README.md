ТЕКСТОВЫЕ КАНАЛЫ

Приложение реализует текстовые каналы наподобие, как в discord, разбитые на разные тематики. Пользователь регистрируется → авторизуется → и может оставлять заметки в виде текста и гиперссылок. Сделано для удобства обучения, в частности для меня, для обучения программированию. Ведь информацию сложно структурировать в голове, когда одно сплошное полотно ссылок и текста вместо множества тематических каналов.
Приложение создано для того, чтобы не потерять полезные ссылки и текстовую информацию. А также, чтоб, остановившись на каком-то месте, продолжить изучать в другой день.
